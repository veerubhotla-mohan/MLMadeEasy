1. Environment Setup

Python is the programming language used throughout. We have many Development Environments to perform the coding. We will walkthrough 3 of them. However, we can use any one of them.

Python

The developers of Python provided an IDE called Python-IDLE to run python scripts. We can download it from here

  1. Head to the 'Downloads' section and download the setup file for your OS
  2. Run the setup file. Make sure to check the box "Add Python to PATH" and continue with the installation
  3. After the installation is done open Python IDLE and start coding!!!
Important: The libraries used for Machine Learning are not readily installed in Python. We need to install them manually. For that, open Command Prompt(Windows) or terminal(Linux/macOS) and install the packages using 'pip' command. Some examples:
  1. pip install numpy
  2. pip install pandas
  3. pip install matplotlib
  4. pip install sklearn

Jupyter Notebook

Jupyter is developed by Anaconda, a popular Data Science platform. We can use Anaconda platform and download Jupyter notebook. Go to this link and download Anaconda. The installation is straigh forward. Go ahead and install.

  1. Now open Anaconda Navigator. Various applications will be available. Find 'Jupyter Notebook' and click on 'Launch'. If it is showing "Install" instead of 'Launch', go ahaed and install. After the installation, you'll see 'Launch'.
  2. Alternatively, we can use 'Spyder' as well.
Important: Anaconda is designed for Data Science and Machine Learning. However, few packages are not readily installed except few(for example tensorflow). Most of the packages are installed. To install a package which is not installed, follow these steps.
  1. Open 'Environments' in the Anaconda Navigator.
  2. A default Environment named 'base(root)' will be available. We can create as many environments as we may wish. If we want to install new package, it is better not to do it using the base Environment. Create a new Environment.
  3. Select the newly created Environment. Change the option in drop-down from 'Installed' to 'Not Installed'.
  4. Search for the package you want to install in the Search bar and install them.
  5. Now go to the Home of the Navigator and change the 'Applications on' tab from base to the newly created Environment
  6. Install Jupyter Notebook again in this Environment and the newly installed packages can be used now.

Google Colab

Colab is Google's powerful IDE for Machine Learning. There is no need of any installation to use Google Colab. All you need is a Gmail account, Google Drive and a browser.

Open a browser, login to your gmail account and go to this link. That's it!!!

It is recommended to use Google Colab since it involves no installation. Everything is readily available to use.
